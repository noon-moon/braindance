Obsidian is a local-first markdown knowledge base application. Notes are plain `.md` files stored on disk, linked together with `[[wikilinks]]`. Each vault is a directory: Obsidian reads and writes files directly, with no proprietary format. Configuration lives in `.obsidian/` at the vault root — `app.json` (application settings), `core-plugins.json` (enabled core plugins), `templates.json` (Templates plugin config), and `workspace.json` (per-machine window layout, typically gitignored). Community plugins extend the core with additional functionality and store their configs in `.obsidian/plugins/`.
