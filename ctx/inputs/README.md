# inputs

**Non-persisted inputs for consideration.**

Drop files here that an agent should take into account for a specific task —
screenshots, exports, clippings, data dumps. Everything in this directory is
**gitignored and ephemeral** (only this README is tracked).

Treat what lands here as context for the work at hand, not as canonical record:

- Read it, use it, but don't rely on anything here persisting.
- If something is worth keeping, **distill it into a vault note** in
  `ctx/vault/` rather than pointing back at the input file.

This is distinct from the vault's own `assets/` and `attachments/`, which *are*
persisted and are embedded in notes via `![[...]]`.
